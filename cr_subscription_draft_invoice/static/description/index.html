Overview

In Odoo 16 and above, invoices for subscriptions are automatically posted as soon as they are generated. While this helps speed up billing, many businesses still prefer to first review their invoices in draft mode before confirming and sending them to customers. This is especially useful to double-check details like customer information, subscription lines, discounts, or taxes, and make corrections if needed.

Our Subscription Draft Invoice module brings back this flexibility. By simply enabling the Draft Invoice option in your subscription plan, you can ensure that invoices are created in draft state instead of being automatically posted. This gives you the control to validate, edit, or confirm them at the right time. Itâ€™s a simple, practical solution to make subscription billing safer and more reliable for your team.

Subscription Draft Invoice

  • In the Subscriptions module, navigate to Configuration > Recurring Plans. Create a new recurring plan or select an existing one.
  • In the recurring plan form view, you will find the Draft Invoice checkbox. Enable this option if you want invoices to be created in draft state instead of being posted automatically.
  • Now navigate to the Products menu and create a new product.
  • Ensure the Subscription option is enabled, then go to the Recurring Prices tab.
  • In the Recurring Prices tab, add the Recurring Plan where the Draft Invoice option is enabled, and set the Recurring Price.
  • Navigate to the Subscriptions menu and create a new Subscription.
  • Select the Subscription Product and the corresponding Recurring Plan, then confirm the quotation.
  • Invoices will be generated automatically according to the recurring period. You can view them by clicking the Invoices option above the subscription form.
  • Here, the Invoice will be created in Draft State. You can review it, make any necessary changes, and then confirm and post the invoice.
  • If you want invoices to be posted automatically again, simply untick the Draft Invoice option in the Recurring Plan form.
  • From now on, all generated invoices will be in Posted state automatically.
