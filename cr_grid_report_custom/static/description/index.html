Overview

The Custom Grid Report module enhances the Sales Order PDF report in Odoo by improving the way product variants are displayed. In standard Odoo, when products are shown in a grid (matrix) format, the same products are also listed again as separate order lines, causing unnecessary repetition and making the report look cluttered. This module solves that problem by ensuring that when a product is already displayed in a matrix table, its duplicate order line is hidden.

In addition, the module introduces three useful headings : Product Description, Total Quantity, and Subtotal at the top of every matrix table. This makes the report more informative, easy to read, and professional. For products that are not displayed in a matrix format, they will continue to appear as normal order lines, ensuring compatibility with all order types. With this module, sales order reports become cleaner, more structured, and customer-friendly, giving businesses a polished and organized document to share with their clients.

Custom Grid Report

  • In the Sales Order, navigate to Configuration > Settings and locate the option Variant Grid Entry.
  • Ensure that this option is enabled to allow products to be displayed in a grid (matrix) format.
  • Ensure that SALES VARIANT SELECTION is set to Order Grid Entry for products that have variants and need to be selected in a grid format.
  • You can now select the variants of the product in a grid format.
  • In the Sales Order PDF report, you will see Description, Total Qty, and Subtotal displayed above grid matrix table.
  • If there are multiple product variant matrix tables, Description, Total Qty, and Subtotal will be displayed above each table.
  • Product lines will only appear if the product does not have a variant matrix table in the report.
  • This ensures a clean report and avoids repetition of products appearing in both the matrix table and the product lines below.

What is the purpose of the Custom Grid Report module?

  • The Custom Grid Report module enhances the Sales Order PDF in Odoo by displaying product variants in a clean grid (matrix) format. It removes duplicate product lines, adds helpful headings like Description, Total Qty, and Subtotal, and ensures the report looks professional, structured, and customer-friendly.

Why don’t I see duplicate product lines when using the grid format?

  • The module automatically hides standard order lines for products that are already displayed in a matrix table. This avoids unnecessary repetition and makes the report much easier to read.

Does this module support quotations as well as confirmed sales orders?

  • Yes. The improved grid format applies to both quotations and confirmed sales orders, ensuring consistency across all customer documents.

Is this app compatible with Odoo Community as well as Enterprise?
