Overview

Managing deliveries to unique addresses is often a challenge in industries like construction, where goods are frequently shipped to temporary locations such as building sites. In standard Odoo, setting a different delivery address typically requires creating a new contact for each location, which can lead to a cluttered and disorganized contact list. This process becomes inefficient when dealing with one-time delivery addresses that don't need to be stored permanently in the database.

The Custom Delivery Address module solves this problem by allowing users to enter custom delivery address details directly on the Sales Order, Invoice, and Delivery Order, without creating a separate contact. New address fields are added to the sales order form, which are auto-filled using the partnerâ€™s delivery address but can be edited freely as needed. This gives businesses the flexibility to handle unique delivery scenarios while keeping the contact list clean and manageable.

  • When a user creates a new Sales Order, the customer's default delivery address automatically appears below the customer field.
  • The user can edit the default shipping address if they wish to specify a different delivery address for that particular order.
  • The user-defined shipping address is saved with the order.This updated address is also shown in the Sales Order PDF report.
  • The updated delivery address is automatically transferred to the Delivery Order. Users can view the custom address directly on the Delivery Order form.
  • The custom delivery address is also displayed in the Delivery Order PDF report. This ensures accurate address details for logistics and delivery teams.
  • The updated delivery address is also transferred to the Invoice record. This helps maintain consistency across the entire sales and billing process.
  • The custom delivery address is displayed in the Invoice PDF report. This ensures clarity for customers and accurate documentation for records.
