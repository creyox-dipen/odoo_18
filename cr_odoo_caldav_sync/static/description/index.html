Which CalDAV providers are supported?

Our module is universally compatible with all major providers including iCloud, Google Calendar, Zoho, and Nextcloud, as well as custom servers like Radicale.

Does it handle recurring series exceptions?

Yes. The module features an advanced recurrence engine that correctly synchronizes both master series and individual modified occurrences (exceptions) without data corruption.

Is the data flow bidirectional?

Absolutely. Updates made in Odoo are instantly pushed to CalDAV, and any changes on your external devices are pulled back into Odoo automatically via background tasks.
