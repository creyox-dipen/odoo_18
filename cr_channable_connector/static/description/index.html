Overview

The Channable Connector for Odoo provides a seamless integration between Odoo and the Channable marketplace platform. Automate your marketplace order lifecycle by importing orders directly into Odoo, synchronizing shipping statuses, and managing cancellations. It ensures your multi-channel sales data remains accurate and up-to-date.



Hot Features

On-Demand Order Import

Sync and import marketplace orders from Channable into Odoo sale orders using a flexible, date-filtered sync wizard.

Shipping & Tracking Sync

Automatically synchronize shipping statuses, carrier details, and tracking numbers from Odoo to Channable.

Interactive Kanban Dashboard

Track live statistics, monitor pending order metrics, and toggle the custom Dark Mode interface directly.

XML Product Feed Engine

Generate dynamic, secure XML product feeds designed exclusively for Channable, with custom product attribute and category tag mappings.

Dynamic Carrier Mapping

Map Channable transporters and carriers dynamically to matching Odoo delivery methods on import.

Auto Customer Lookup

Intelligently lookup matching customers by email, or auto-create Odoo contact records.

Billing & Order Automation

Configure settings to auto-validate quotations, confirm sales orders, create invoices, and register payments.

Cancellation & Refund Sync

Keep order states in sync automatically with auto-generation of credit notes and refund handling.

Multi-Marketplace Support

Manage multiple Channable connections, projects, and marketplaces/channels from a single Odoo database instance.

Centralized Sync Logs

Access detailed synchronization history and track exceptions with automatic error assignment to specific Odoo users.

Smart Direct Redirect

Access Channable order details directly via smart links embedded directly on Odoo Sales Orders.

Sync Progress Monitor

Real-time progress bars and synchronization indicators represent processing status during batch runs.

  • FAQ

  • Create a Channable Connection. The connection acts as the primary bridge using your Channable API credentials. Navigate to Sales > Configuration > Connections.
  • Define Channable Projects
    Go to Sales > Configuration > Projects.
  • Add the Unique ID for this project from Channable.
  • Now Configure Marketplaces
    Go to Sales > Configuration > Marketplaces.
  • You can also view the marketplace dashboard in Kanban view, where you can see the number of orders pending validation and shipment. The dashboard also allows you to switch between dark and light themes and provides quick access to synchronization actions.
  • Order Configuration ID: The unique ID for the specific marketplace channel in Channable.
  • Auto-Validation: Enable toggles to automatically confirm Quotations, create Sales Orders, and validate Invoices.
  • Import Order Status: Select the order statuses that you want to import from Channable. These statuses will be automatically selected in the synchronization wizard, and orders with the selected statuses will also be imported when the scheduled synchronization runs.
  • Now you can sync orders by clicking on button sync orders.
  • In the popup wizard, select a Start Date and End Date to import orders within a specific date range. You can also modify the selected Import Order Status values before starting the synchronization.
  • Alternatively, enable Import Orders by ID if you want to import specific orders. Enter a comma-separated list of Channable Order IDs, then click Import Orders to start the import process.
  • If any errors occurred during the sync, you can check it.
  • Now you can check the successful orders are created inside the Odoo.
  • You can cancel an order from Odoo and click Cancel in Channable. The order will also be cancelled in Channable, if it has not already been shipped.
  • If you validate a delivery in Odoo for an order that is not yet marked as shipped, the order will be automatically marked as shipped and the tracking number will be updated in Channable.
  • From the Sale Order, you can navigate directly to the related Channable Order by clicking on Channable Order smart button.
  • By clicking Sync Order sale order form, you can synchronize the order details from Channable to Odoo. By clicking Sync State, you can synchronize the order status from Channable to Odoo. If an order is cancelled in Channable and you perform a sync, the order will also be cancelled in Odoo.
  • You can also configure product fields and attributes with custom XML tags under the Attribute Mappings section of the marketplace. These mappings are used when exporting product data from Odoo to Channable.
  • copy the Product Feed URL to export the products to Channable.
  • Now navigate to Setup >> Import and select the XML option in channable.
  • Paste the product feed URL into the XML file url field then click on Run Import button.
  • You can see the mapping of standard product fields and custom fields along with the custom XML tags you configured. Map these fields to the corresponding Channable fields and then import the products.
  • You can also configure shipping method mappings with Odoo delivery carriers. When an order is synchronized, the carrier mapped to the corresponding Channable shipping method will be automatically assigned to the sale order.
