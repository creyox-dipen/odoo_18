Overview

The Stripe Refund Payment module for Odoo allows businesses to process refunds for payments made via Stripe directly from the Odoo backend. Users can quickly initiate either full or partial refunds without leaving Odoo, eliminating the need to manually log in to Stripe. Once the refund is triggered, the module ensures it is securely processed and linked to the original payment, keeping your financial records organized and consistent.

The module also automatically generates credit notes for refunded amounts and syncs all refund information between Stripe and Odoo. This reduces manual effort, prevents errors, and gives businesses a fast, professional, and reliable way to manage refund requests efficiently within Odoo.

Stripe Refund Invoice Payment

  • First, configure the Stripe Payment Provider in Odoo.
  • To do this, go to Invoicing >> Configuration >> Payment Providers.
  • There, locate the Stripe Payment Provider and click on it.
  • Enable the Stripe Payment Provider and enter the required Publishable Key and Secret Key to connect with Stripe.
  • Click on Generate Webhook to automatically create the webhook endpoint in Stripe.
  • When you open a Paid Invoice, you will see the Refund (Stripe) button available on the invoice form.
  • By clicking the button, a popup will open where you can select the Transaction and enter the Refund Amount.
  • If there is only one transaction linked to the invoice, the Transaction will be auto-selected, and the full refund amount will be pre-filled.
  • You can adjust the refund amount to create a Partial Refund, if required.
  • A Notification Popup will appear confirming that the refund request has been successfully sent to Stripe.
  • A Refund Record will be created and displayed under the related transaction form.
  • You can view complete details by clicking on the Refunds smart button.
  • A Credit Note will be automatically generated for the refunded amount.
  • The user can view more details about the Refund Process directly from the Stripe Dashboard.

Can I make partial refunds in Odoo with this module?

  • Yes. By default, the full refund amount is selected, but you can change the amount to process a partial refund.

How do I track refunds in Odoo?

  • After refunding, you can see a refund record in the transaction form and also check details through the Refund smart button.

Can I see multiple refund records for the same invoice?

  • Yes. If you process multiple partial refunds, each one will be stored and visible in the Refund Records section.

Is this app compatible with Odoo Community as well as Enterprise?

Is this app compatible with Odoo Online?
